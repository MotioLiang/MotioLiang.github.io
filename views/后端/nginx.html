            deepin15.9安装nginx

            前端日记 2018-12-15 nginx

            # 使用wget命令下载

            wget -c https://nginx.org/download/nginx-1.15.0.tar.gz
            1

            # 配置

            ./configure
            1

            # 编译安装

            # 1.安装

            make
make install
            1
            2

            # 2.查看安装路径

            whereis nginx
            1

            # 启动停止nginx

            cd /user/local/nginx/sbin/
./nginx
./nginx -s stop
./nginx -s quit
./nginx -s reload
            1
            2
            3
            4
            5

            ./nginx -s quit:此方式停止步骤是待nginx进程处理任务完毕进行停止。 ./nginx -s stop:此方式相当于先查出nginx进程id再使用kill命令强制杀掉进程。

            # 查询nginx进程

            ps aux|grep nginx
            1

            # 重启nginx

            # 1.先停止再启动（推荐）：

            对 nginx 进行重启相当于先停止再启动，即先执行停止命令再执行启动命令。如下：

            ./nginx -s quit
./nginx
            1
            2

            # 2.重新加载配置文件：

            当 ngin x的配置文件 nginx.conf 修改后，要想让配置生效需要重启 nginx，使用-s reload不用先停止 nginx 再启动 nginx 即可将配置信息在 nginx 中生效，如下：

            ./nginx -s reload
            1

            # 开机自启动

            即在rc.local增加启动代码就可以了。

            vi /etc/rc.local
            1

            增加一行 /usr/local/nginx/sbin/nginx 设置执行权限：

            chmod 755 rc.local
            1
